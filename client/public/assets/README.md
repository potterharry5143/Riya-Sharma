# /public/assets/

Place your images here. All images should ideally be in **WebP** format for best performance.

## Expected files:

| Filename            | Description                        |
|---------------------|------------------------------------|
| priya-photo.webp    | Profile/intro image shown at top of card |
| work-1.webp         | First artwork/project thumbnail    |
| work-2.webp         | Second artwork/project thumbnail   |
| work-3.webp         | Third artwork/project thumbnail    |

Add more work images as needed and reference them in:
  `client/src/chatbot/specialResponses.js`

## How to convert images to WebP (free tools):
- https://squoosh.app  (drag & drop, choose WebP, download)
- https://convertio.co/jpg-webp/
- CLI: `cwebp input.jpg -o output.webp`
